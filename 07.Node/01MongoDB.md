# 数据库

## 1.数据库简介

数据库Database，是按照数据的结构来组织、存储和管理数据的仓库。

在程序开发中，经常会有很多数据需要保存，比如用户的信息，理论上，可以将数据保存到一个对象变量里，但是变量都是保存在内存中的，一旦程序运行结束或者计算机断电时，程序中运行的数据就会丢失。

所以我们需要将程序运行的数据持久的存在磁盘里，以确保数据的安全性。

我们也可以考虑将数据保存在文件系统里，但是不太方便，我们使用数据库专门来存储数据。

## 2.数据库的分类

- 关系型数据库(RDBMS)：
  - MySQL、SQL Server...
  - 关系数据库里都是表
  - SQL是一门语言
- 非关系型数据库(NO SQL)：
  - MongoDB、Redis...
  - 键值对数据库
  - 文档数据库MongoDB
